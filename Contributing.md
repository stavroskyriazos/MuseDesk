# 🤝 Contributing to Museum Helpdesk System

Thank you for contributing to this project!
This project follows a collaborative and structured development approach using GitHub.

---

## 👥 Team Members

* Afroditi: Login & Register API, Login & Register UI, Dashboard API & UI, Database
* Stavros: Zammad & App Set Up, Documentation, Ticketing API & UI

Both members collaborate on testing and debugging of the project.

---

## 🧠 Contribution Philosophy

* Work collaboratively and communicate regularly
* Keep code clean, simple, and well-structured
* Ensure all changes are tested before merging
* Maintain clear documentation for all features

---

## 🧪 Testing

* Test all features before submitting changes
* Ensure no existing functionality is broken

---

## 📝 Reporting Issues

* Use GitHub Issues to report bugs or suggest features
* Provide clear descriptions and steps to reproduce
